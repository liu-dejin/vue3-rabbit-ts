# vue-rabbit-ts

使用 TypeScript 重写的黑马小兔鲜项目，完善了视频没有讲到的一些功能，仅供学习参考。

## 项目介绍

本项目是基于黑马程序员《Vue3 小兔鲜》课程视频的 TypeScript 重写版本。在原有功能基础上，使用 TypeScript 进行了类型增强，并完善了一些视频中未详细讲解的功能。

## 视频教程

B站视频教程：[Vue3 小兔鲜](https://www.bilibili.com/video/BV1Ac411K7EQ?spm_id_from=333.788.player.switch&vd_source=76df1e14a7fc9c2ecb39d2923a8a320e&p=99)

## 技术栈

- Vue 3
- TypeScript
- Vite
- Pinia

- Element Plus
- Axios

## 项目特点

- 完整的 TypeScript 类型定义
- 完善了视频未讲解的功能模块
- 代码结构清晰，易于维护
- 适合 Vue3 + TypeScript 学习参考

## 如何运行项目

### 1. 安装依赖

```bash
pnpm install
```

### 2. 启动开发服务器

```bash
pnpm dev
```

启动后，在浏览器中访问 `http://localhost:5173` 即可查看项目。

### 3. 构建生产版本

```bash
pnpm build
```

### 4. 代码检查

```bash
pnpm lint
```
